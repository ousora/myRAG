## 当前架构（清理后）

```
myrag/
├── config.py              # 配置加载器
├── pipeline.py            # Facade: Chunker → chunkers/
├── parsers/
│   ├── dispatcher.py      # resolve_parser() — MarkItDown + Trafilatura
│   └── text_cleaner.py    # TextCleaner (YAML config support)
├── cleaners/              # → parsers.text_cleaner (backward compat)
├── chunkers/              # LangChain MarkdownHeaderTextSplitter + RecursiveCharacterTextSplitter
├── formatters/
│   ├── __init__.py        # format_text_async() — LLM endpoint
│   ├── prompts.py         # System prompt template
│   └── writer.py          # write_to_md() — H2-H5 rendering
├── embedders/
│   └── bge_m3.py          # Embedder (OpenAI-compatible API)
├── storage/
│   └── sqlite_vec.py      # SQLiteVecStore — integrated via process_file_hybrid(store_path=...)
├── conf/
│   ├── config.yaml        # User instance config (gitignored)
